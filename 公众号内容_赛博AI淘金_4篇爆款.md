## 🎯 文章3：教程类

**标题：掌握这10个DeepSeek提示词公式，普通人也能写出10万+爆文**

**封面图建议**："10个Prompt公式" + 爆文截图

---

### 正文：

很多兄弟问我：为什么同样的AI工具，别人写出来的东西那么专业，我写的就像小学生作文？

答案只有一个：**你不会写提示词（Prompt）。**

今天我把珍藏的10个DeepSeek提示词公式全部分享出来，亲测有效，直接复制就能用。

---

### **公式1：身份+任务+约束**
```
你是一位有10年经验的[行业专家]，
请帮我[具体任务]，要求：
1. [约束条件1]
2. [约束条件2]
3. [约束条件3]
```

**示例：**
你是一位有10年经验的小红书运营专家，
请帮我写一篇关于"AI绘画副业"的爆款笔记，要求：
1. 标题要有数字和情绪词
2. 正文分点说明，每点有emoji
3. 结尾引导评论互动

---

### **公式2：背景+问题+目标+输出格式**
```
【背景】[描述当前情况]
【问题】[遇到的具体问题]
【目标】[想要达到的效果]
【输出】请以[格式]呈现
```

**示例：**
【背景】我是一个AI工具博主，有5000粉丝
【问题】最近内容互动率下降，粉丝增长停滞
【目标】希望提高内容质量和粉丝粘性
【输出】请以清单形式给出10个优化建议

---

### **公式3：角色扮演+场景设定**
```
假设你是[具体角色]，正在[具体场景]。
请针对[目标用户]的需求，给出[具体建议]。
语气要[风格描述]。
```

**示例：**
假设你是马云，正在给创业者分享经验。
请针对25-35岁想搞副业年轻人的需求，给出AI时代的机会建议。
语气要接地气、有激情、多用口语。

---

### **公式4：步骤拆解法**
```
请帮我完成[任务]，按以下步骤：
第1步：[步骤1]
第2步：[步骤2]
第3步：[步骤3]
...
最后给出完整方案。
```

**示例：**
请帮我做一个AI副业变现方案，按以下步骤：
第1步：分析当前热门AI副业方向
第2步：评估每个方向的投资成本和收益潜力
第3步：根据0基础、低成本的标准筛选最优方案
第4步：给出具体的7天执行计划
最后给出完整方案。

---

### **公式5：对比分析法**
```
请对比分析[A]和[B]，从以下几个维度：
1. [维度1]
2. [维度2]
3. [维度3]
最后给出适合[场景]的推荐。
```

**示例：**
请对比分析DeepSeek和ChatGPT，从以下几个维度：
1. 中文写作能力
2. 代码生成能力
3. 使用成本
4. 适合的场景
最后给出个人自媒体运营者的推荐。

---

### **公式6：内容改写公式**
```
请将以下内容改写成[平台]风格：
原文：[粘贴原文]
要求：
- [要求1]
- [要求2]
- [要求3]
```

**示例：**
请将以下内容改写成小红书风格：
原文：人工智能技术的发展为内容创作带来了革命性变化...
要求：
- 口语化、接地气
- 每句加emoji
- 分点说明
- 结尾引导点赞收藏

---

### **公式7：爆款标题生成**
```
请根据关键词"[关键词]"，生成10个爆款标题。
要求：
- 使用数字
- 制造好奇或恐惧
- 字数在20字以内
- 符合[平台]风格
```

---

### **公式8：数据驱动分析**
```
我有以下数据：
[粘贴数据]
请分析：
1. 数据反映了什么问题？
2. 可能的原因是什么？
3. 给出3个优化建议
```

---

### **公式9：AIDA营销文案**
```
请用AIDA模型（注意-兴趣-欲望-行动）写一段文案，
推广[产品/服务]。
目标用户：[用户画像]
核心卖点：[卖点]
```

---

### **公式10：反向提问法**
```
我想[目标]，但不知道怎么做。
请向我提问，了解我的情况后给我建议。
```

**示例：**
我想通过AI工具做副业月入5000，但不知道怎么做。
请向我提问，了解我的情况后给我建议。

---

### **使用技巧：**

1. **不要一次给太多要求** - 分步骤来，效果更好
2. **不满意就让它重写** - 直接说"重写，更XXX一点"
3. **多用举例** - 给AI一个例子，它学得更快
4. **保存常用Prompt** - 建个文档，好用的直接复制

---

**记住：Prompt的质量决定了AI输出的质量。**

把这10个公式收藏起来，下次用DeepSeek的时候直接套。用熟了，你也能写出10万+爆文。

觉得有用的点个在看，转发给需要的朋友。
